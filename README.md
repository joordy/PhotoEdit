## 🔑 Installation

### Clone the repository

```bash
  git clone https://github.com/joordy/web-app-from-scratch-2021.git
```

### Navigate to the repository and nstall the packages

```bash
  npm install
```

### Start local dev environment

```bash
  npm run dev
```

### Build export

```bash
  npm run build
```
